<br>
 
## 文档开发指引 :point_right:

1. 在docs/document 文件夹下创建文档目录 例如： docs/document/vue

2. 在创建的目录下创建你想要分享的知识点.md文件  例如：vueLifeCycle.md

3. 按照markdown规范编写文档

> 注意: 如果文档是引用网络文档（别人的博客、个人站点技术文档等），请注明出处

4. 扩展docs/.vuepress/config.js, 为你的文档配置路由
